# rafaelctorres.dev

Personal portfolio and blog — documenting my work in applied mathematics for multimedia, deep learning, and web development.

Built with Astro, Tailwind CSS, and TypeScript.

## Stack

- [Astro](https://astro.build) — static site framework with MDX support
- [Tailwind CSS](https://tailwindcss.com) — utility-first styling
- [TypeScript](https://www.typescriptlang.org) — strict mode
- [Space Grotesk](https://fonts.google.com/specimen/Space+Grotesk) + [JetBrains Mono](https://www.jetbrains.com/lp/mono/)

## Commands

| Command         | Action                          |
| :-------------- | :------------------------------ |
| `npm install`   | Install dependencies            |
| `npm run dev`   | Start dev server at localhost:4321 |
| `npm run build` | Build for production to ./dist/ |
| `npm run preview` | Preview production build      |
